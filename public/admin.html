Cómo usar una API Key
Opción 1 — Header HTTP (recomendado):
X-Api-Key: tu_api_key_aqui

Opción 2 — Query parameter:
GET /tabla?api_key=tu_api_key_aqui

Ejemplo con curl:
curl -H "X-Api-Key: l3ho_xxxx" https://tu-dominio/tabla
